# VivahaLuxe Built-in Music Library (`/public/music`)

You can drop any custom wedding audio tracks (`.mp3`, `.wav`, `.m4a`) directly into this folder!
Once added, they can be referenced by path (e.g., `/music/shehnai-royal.mp3`) or selected directly from the Built-in Music Library dropdown in the Couple Studio (`/admin`).

### Suggested Naming:
- `royal-piano.mp3`
- `shehnai-flute.mp3`
- `sufi-melody.mp3`
- `veena-classical.mp3`
